## Bot de promoções

> Bot de promoções para WhatsApp com curadoria manual, ofertas coletadas do site da Amazon Brasil.
> Coleta ofertas em tempo real, gera copy persuasivo via IA e distribui no WhatsApp com aprovação humana antes do envio, reduzindo risco de ban e garantindo qualidade editorial.
